# Water Quality Prediction App

This project runs a local Dash app for water quality prediction.

## Requirements

- Python 3
- A local virtual environment in `venv/`

## How to Run

Open a terminal in the project folder and run these commands in order:

```bash
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

After the app starts, open your browser and go to:

```text
http://127.0.0.1:8050
```

## Notes

- The app runs locally on port `8050`.
- If `venv/` does not exist yet, create it first with `python3 -m venv venv`.
